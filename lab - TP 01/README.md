#  TP01 – Python Web Scraping Case Study

##  الهدف من المشروع
يهدف هذا المشروع إلى تعلم أساسيات **استخلاص البيانات من الويب (Web Scraping)** باستخدام لغة **Python** ومكتباتها الشهيرة مثل:
- `requests` لجلب صفحات الويب.
- `BeautifulSoup` لتحليل واستخلاص العناصر من كود HTML.
- `pandas` لتنظيم البيانات وتخزينها في ملفات CSV.

الموقع المستخدم للتجربة هو:
> [https://books.toscrape.com](https://books.toscrape.com)  
وهو موقع تعليمي مخصص للتدريب على تقنيات Web Scraping.
